# flag-game

Nology project to create a game of my choice.

--AIMS

1. Create a working game.
2. Practice using git and github.
3. Apply what you have learnt.

--REQUIREMENTS

1. Pseudocode
2. Public Repository with a readme
3. At least 25 commits.
4. JS code must be formatted as functions or classes.
5. You must use click or keypress events in JavaScript to trigger event in the game.
6. Use mobile first approach.
7. Add a link to this project on your portfolio.

--PSEUDOCODE

1. Array of objects with country, flag, continent and latitude and longitude inside.
2. Math.random to render a random flag.
3. Input box which autofills countries.
4. Begin with correct/incorrect, then correct/incorrect continent, then distance from correct answer, then arrow to show direction from the correct answer and percentage.
5. Refresh button which starts the game again.
6. Pop-up(alert) that tells you the correct answer.
7. Pop-up(alert) that keeps track of the amount of games you have won.
8. Place 5 squares over the flag using relative and absolute which are taken away after each answer.
